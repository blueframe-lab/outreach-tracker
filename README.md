# Outreach Tracker

動画制作・SNS運用代行の営業候補を管理するための小さなWebアプリです。営業先のURL、連絡方法、ステータス、優先度、メモを保存し、検索・絞り込み・編集・削除ができます。

## 機能

- 営業先の登録、編集、削除
- 営業先名、種別、URL、連絡方法、ステータス、優先度、メモの管理
- カード形式の一覧表示
- キーワード検索
- ステータスと優先度での絞り込み
- localStorageへの自動保存
- 初回表示用のサンプルデータ3件
- PC、スマホ両対応のレスポンシブUI

## 技術構成

- TypeScript
- React
- Vite
- localStorage

外部API、ログイン、データベースは使用していません。

## 起動方法

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いて確認します。

## ビルド

```bash
npm run build
```

ビルド成果物は `dist` に出力されます。
